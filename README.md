# Endless DOSTH Road Survival

A retro-style endless runner game inspired by Chrome Dino, where you navigate a pixelated car through dynamically generated road hazards.

## How to Play

1. Use arrow keys to control your car:
   - ↑ - Accelerate
   - ↓ - Decelerate/Brake
   - ← → - Switch Lanes
   - SPACE - Pause/Resume
